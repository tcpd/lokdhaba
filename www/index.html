Welcome to Lok Dhaba!

Lok Dhaba is a repository of Indian election results - both Vidhan Sabha (state level) and Lok Sabha (national level) - beginning 1962. The data is free and open for anyone to use. We encourage students, researchers, media persons, policy makers to engage with our data. Read what we have published using this data and how its been used by others. Should you have any query and/or notice an error, please send us a message or write us at tcpd-contact@ashoka.edu.in.

We will update the data with the 2019 GE results as and when they come in and are thoroughly checked.

For more information, please see the documentation tab, which includes a detailed codebook containing all the variables in Lok Dhaba. Should you use any information from LokDhaba, please be sure to cite the Trivedi Center for Political Data at Ashoka University.

LokDhaba is a project of the Trivedi Center for Political Data, a research center run by the Political Science and Computer Science Departments at Ashoka University.


LokDhaba's interface has been developed with open source tools such as R, Shiny, Plotly, data tables, rgdal, and Leaflet.


The project lead is Mohit Kumar who is a Data Scientist and GIS Engineer at TCPD.
